# Pytorch-StarGAN-Digits
Unofficial Pytorch implementation of StarGAN for generating Digit-5 datasets (MNIST, SVHN, SynDigits, MNIST-M, and USPS).

### Data layout

    .
    ├── MNIST 
    |	└MNIST Images
    |
    |── MNIST-M
    |	└MNIST-M Images
    |
    |── SVHN  
    |	└SVHN Images
    |
    |── SynDigits
    |	└SynDigits Images
    |
    └── USPS
    	└USPS Images
